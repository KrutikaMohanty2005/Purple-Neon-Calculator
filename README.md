# Purple Neon Calculator

A sleek, responsive calculator web application with a glowing purple neon aesthetic and modern UI.

## Features

- **Safe Evaluation** - No `eval()` vulnerability; uses secure function parsing
- **Keyboard Support** - Full keyboard input (0-9, operators, Enter, Backspace, Escape)
- **Calculation History** - Persistent history stored in localStorage (max 20 entries)
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Smart Input** - Prevents invalid inputs like double operators or multiple decimals
- **Percentage Support** - Quick percentage calculations

## How to Use

### Mouse/Touch
- Click number and operator buttons to build your expression
- Press `=` to evaluate
- Use `AC` to clear all, `DEL` to delete last character
- Click the clock icon to view/clear history

### Keyboard
| Key | Action |
|-----|--------|
| `0`-`9` | Input digits |
| `+` `-` `*` `/` | Operators |
| `%` | Percentage |
| `.` | Decimal point |
| `Enter` / `=` | Calculate |
| `Backspace` | Delete last |
| `Escape` / `Delete` | Clear all |
